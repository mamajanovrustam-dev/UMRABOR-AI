# @umrabor/miniapp

Mini App UMRABOR в Super-App Click (`miniapp.umrabor.uz`).

Будет реализован в **Sprint 4** (React 19 + Vite + Tailwind). Phone shell 380×790px для встраивания в Click, mobile-web fallback для обычного браузера.

См. `prototypes/umrabor_client_v6.html` для референса дизайна и `docs/UMRABOR_Full_Documentation.md` (Часть IV) для спецификации.
