# UMRABOR — оригинальные прототипы (архив)

Это HTML-прототипы v1.0 (21 мая 2026), на основе которых строится production-проект.

| Файл | Продукт |
| --- | --- |
| `umrabor_platform_v1.html` | АРМ Платформы (UMRABOR Inc.) |
| `umrabor_admin_v5.html`    | АРМ Партнёра (тур-операторы) |
| `umrabor_client_v6.html`   | Mini App в Click |
| `umrabor_web_v1.html`      | Веб-сайт (полный SPA) |
| `index.html`               | Индексная страница пакета |
| `umrabor_overview.html`    | Обзорная страница |
| `umrabor_faq.html`         | FAQ-страница |

Прототипы открываются двойным кликом в браузере.

> ⚠️ **Не использовать в production.** Это статичные HTML с JS-моками — без backend, БД, реальной авторизации и интеграций. Хранятся как референс дизайна и поведенческой логики для команды разработки.

Полная спецификация и описание всех экранов: [`docs/UMRABOR_Full_Documentation.md`](../docs/UMRABOR_Full_Documentation.md).
